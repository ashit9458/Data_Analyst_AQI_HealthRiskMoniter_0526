# Real-Time Air Quality & Health Risk Monitor

## Overview
This project predicts AQI health risk zones across Indian cities using live AQI sensor data, Machine Learning, Azure SQL, Power BI, and Tableau.

## Features
- Live AQI ingestion
- Azure Data Factory scheduling
- Azure SQL data storage
- Data cleaning using SQL CTEs
- Feature engineering using Pandas
- Random Forest classification model
- Power BI dashboards
- Tableau storytelling

## Technologies Used
- Python
- Azure SQL
- Azure Data Factory
- Power BI
- Tableau
- scikit-learn

## Machine Learning
Random Forest Classifier predicts:
- Safe
- Moderate
- Hazardous
